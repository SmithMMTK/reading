
_Azure AI Translator_ เป็นบริการบนระบบคลาวด์ที่ช่วยให้แอปพลิเคชันของคุณสามารถ _แปลภาษา_ ได้อย่างมีประสิทธิภาพ บริการนี้เป็นเทคโนโลยี _machine translation_ ที่ผ่านการพิสูจน์แล้วว่า _ปรับแต่งได้_ และ _ขยายขนาดได้ง่าย_ ซึ่งปัจจุบันถูกใช้งานโดยผู้ใช้ทั่วโลกผ่านแอปต่าง ๆ เช่น _Microsoft Office_, _Bing_, และ _Microsoft Edge_

Translator ใช้เทคนิค _Neural Machine Translation_ ที่ช่วยให้ผลลัพธ์มี _ความลื่นไหล (fluency)_ และ _อ่านเข้าใจง่าย (readability)_ โดยไม่แปลคำแยกกัน แต่แปลโดยใช้ _บริบทของทั้งประโยค_

ผู้ใช้งานยังสามารถปรับแต่งการแปลให้เหมาะกับบริบทของตนเองผ่าน _Custom glossaries_ และ _Custom Translation_ ซึ่งเหมาะมากหากเนื้อหาของคุณมี _คำศัพท์เฉพาะองค์กร_ หรือ _สรรพนาม_ ที่ใช้เฉพาะเจาะจง

## Translator services

- _Text-to-text translation_: แปลข้อความแบบเรียลไทม์ รองรับกว่า _90 ภาษา_ และภาษาถิ่นต่าง ๆ ทำให้สามารถเพิ่มประสบการณ์ผู้ใช้แบบหลายภาษา (_multilanguage UX_) ได้ในทุกระบบปฏิบัติการ

- _Document Translation_: รองรับไฟล์ PDF, HTML, .tsv และเอกสาร Microsoft เช่น PowerPoint, Outlook, Excel โดยสามารถแปลเอกสารได้สูงสุด _1,000 ไฟล์ต่อคำขอ_ พร้อมทั้งคง _โครงสร้าง_ และ _รูปแบบ_ ของต้นฉบับไว้

- _Custom Translation_: ปรับแต่งระบบการแปลได้ใน _60 ภาษา_ โดยสามารถเพิ่มคำศัพท์เฉพาะทาง เพื่อให้การแปล _แม่นยำ_ และ _เหมาะกับบริบทของธุรกิจหรืออุตสาหกรรม_ มากขึ้น สามารถฝังระบบนี้เข้าไปในแอป เว็บไซต์ หรือเวิร์กโฟลว์ที่มีอยู่แล้วได้

### More functionality

- _Automatic language detection_: ระบบสามารถตรวจจับ _ภาษาต้นทางอัตโนมัติ_ เพื่อปรับปรุงความถูกต้องและประสบการณ์ผู้ใช้ รองรับมากกว่า _50 ภาษา_ เช่น Klingon หรือ Kurdish
- _Transliteration_: ใช้ถอดเสียงจาก _อักษรหนึ่งไปยังอีกอักษรหนึ่ง_ เช่น การถอดเสียงชื่อหรือคำที่แปลตรงตัวไม่ได้ เช่น "Konnichiwa" จากภาษาญี่ปุ่น
- _Dictionaries_: แสดงคำแปล _ทางเลือก_ สำหรับคำหรือสำนวนต่าง ๆ และคุณสามารถสร้าง _พจนานุกรมเฉพาะของคุณเอง_ ที่ใช้กับระบบแปลได้อัตโนมัติ
- _Profanity filters_: ตัวกรองคำหยาบ หากไม่ตั้งค่า ระบบจะแปลคำหยาบตามปกติ แต่คุณสามารถตั้งค่าให้ _ลบ_ หรือ _แท็ก_ คำเหล่านั้นได้ตามต้องการ

> _ความปลอดภัยระดับองค์กร_

Translator สร้างอยู่บนโครงสร้างพื้นฐานของ _Azure_ จึงมีความปลอดภัย ความพร้อมใช้งาน และการจัดการระดับองค์กร ข้อมูลที่ส่งผ่าน API จะไม่ถูกจัดเก็บไว้ในศูนย์ข้อมูลของ Microsoft
